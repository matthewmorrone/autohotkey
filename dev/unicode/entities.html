HTML Predefined Character Entities

Summary

This table lists the predefined character entities of HTML, derived from chapter 24 of the HTML specification.

(Note that the “&apos;” entity is only available in XHTML and not in plain HTML.)

For each character, the first column gives the Unicode hexadecimal number of the character, the second gives a sample of the character, the third gives the Unicode name of the character, the fourth gives the entity name (as it should be entered in an HTML document) and the fifth gives the equivalent numeric character reference. (Actually, &lt; is not truly equivalent to &#60; nor &amp; to &#38;, but the rest work as expected.)
